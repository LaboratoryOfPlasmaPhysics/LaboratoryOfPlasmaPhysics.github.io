Nomad DevDejs!

Created in 2016, Devdejs are lunchtime meeting for developers. Originally located at LPP, DevDejs will now occur in different labs of Ecole Polytechnique.

Devdejs

Devdejs are weekly lunchtime meetings for developers at Ecole Polytechnique. The idea is to have a time and place to meet, discuss and share about code development techniques, projects, tools related to numerical simulation, data analysis, instrumentation and experiments.

Our code projects

PHARE

Parallel Hybrid Particle-In-Cell code with Adaptive mesh REfinement. Modern C++ code in development with support of the labex Plas@Par. See in GitHub

LPPIC

2.5D Parallel electrostatic Particle-In-Cell code developed at LPP for studying plasma propulsors.

SciQLOP

Efficient and Ergonomic application to explore spacecraft in situ plasma data and create catalogs of events. See on GitHub

SocExplorer

SocExplorer is an open source generic System On Chip testing software/framework. See on GitHub
